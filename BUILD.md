# Flagit React SDK

Simple test to verify the package structure.

## Files created:

1. package.json - Package configuration and dependencies
2. tsconfig.json - TypeScript configuration 
3. vite.config.ts - Vite build configuration
4. src/types.ts - TypeScript type definitions
5. src/client.ts - Core Flagit client
6. src/context.tsx - React context and provider
7. src/hooks.ts - React hooks for feature flags
8. src/components.tsx - React components
9. src/index.ts - Main export file
10. README.md - Comprehensive documentation
11. .gitignore - Git ignore file
12. example.tsx - Usage examples

## To use this SDK:

1. Install dependencies: `npm install`
2. Build the library: `npm run build`
3. The built files will be in the `dist/` folder
4. You can then publish to npm with: `npm publish`
